Welcome to Video App built with ReactJS in ES6.

This is a Video Player App built with ReactJS, written in ES6 or ES2016, and transpiled with the help of Babel and WebPack.

Clone this Repo and let's get started!

Very simple: 

  > git clone git@github.com:zainxyz/VideoAppReact.git
  > cd VideoAppReact
  > npm install
  > npm start
